## Chess Board Spectral Analysis: Mapping the Knight's Influence
This project applies Spectral Graph Theory to the game of chess to identify the most influential squares for a Knight. 
By treating the chessboard as a graph and legal moves as edges, we use the Principal Eigenvector of the board's adjacency matrix to generate a 
"Power Map" of the Knight's connectivity.
